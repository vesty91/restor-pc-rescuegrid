OUTILS WINPE - Restor-PC RescueGrid
===================================
Les dossiers smartctl / testdisk / photorec sont VIDES pour l'instant.

Pour ajouter un logiciel :
1. Copiez un portable .exe (ou son dossier) ICI
2. Rebootez WinPE : un bouton apparait sur le bureau

Test : lancez Notepad.cmd / Invite-CMD.cmd depuis le bureau WinPE.

Attention : beaucoup d'applis Windows classiques NE MARCHENT PAS en WinPE
(DLL / .NET Desktop manquants). Preferer des outils prevus pour WinPE.

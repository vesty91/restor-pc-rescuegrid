Centre outils technicien

Clé USB RescueGrid, menu WinPE, PXE réseau et déploiement Synology.

1. Créer clé USB

Prépare une clé technicien avec scripts RescueGrid, menu WinPE et outils externes.

powershell -ExecutionPolicy Bypass -File agent\windows\Create-RescueGridUSB.ps1 -TargetDrive E: -DashboardUrl "http://192.168.1.10:8000" -IncludeProject

2. Menu WinPE

Lance le menu atelier depuis la clé ou depuis WinPE.

powershell -ExecutionPolicy Bypass -File agent\windows\Start-RescueGrid.ps1

3. Serveur PXE

Boot réseau pour démarrer plusieurs machines sans clé USB.

powershell -ExecutionPolicy Bypass -File agent\windows\Setup-PXERescueServer.ps1

4. Déploiement Synology

Docker Compose avec PostgreSQL, Nginx et MinIO sur NAS Synology.

docker-compose -f docker-compose.synology.yml up -d

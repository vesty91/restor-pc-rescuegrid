Centre outils technicien

Clé USB RescueGrid, menu WinPE, PXE réseau et déploiement Synology.

1. Créer clé USB

Prépare une clé technicien avec scripts RescueGrid, menu WinPE et outils externes. Script unique depuis v12.3 (remplace Build-RescueGridUSB.ps1).

powershell -ExecutionPolicy Bypass -File agent\windows\Create-RescueGridUSB.ps1 -TargetDrive E: -DashboardUrl "http://192.168.1.10:8000" -IncludeProject

Voir docs/TECHNICIAN_MANUAL.md pour le détail (formatage FAT32/NTFS, copie WinPE si ADK installé).

2. Construire l'image WinPE (ADK requis)

Génère automatiquement boot.wim (+ ISO optionnelle) avec PowerShell/Scripting intégrés, depuis un Windows ADK + add-on WinPE installés localement. À exécuter en administrateur.

powershell -ExecutionPolicy Bypass -File agent\windows\Build-RescueGridWinPE.ps1 -Force -BuildIso

Puis relancer l'étape 1 avec -WinPEBasePath C:\WinPE pour copier boot.wim sur la clé.

3. Menu WinPE

Lance le menu atelier depuis la clé ou depuis WinPE.

powershell -ExecutionPolicy Bypass -File agent\windows\Start-RescueGrid.ps1

4. Serveur PXE

Boot réseau pour démarrer plusieurs machines sans clé USB.

powershell -ExecutionPolicy Bypass -File agent\windows\Setup-PXERescueServer.ps1

5. Déploiement Synology

Docker Compose avec PostgreSQL, Nginx et MinIO sur NAS Synology.

docker-compose -f docker-compose.synology.yml up -d
